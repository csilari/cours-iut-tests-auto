### Architectural Style

Starting from checkpoint4, the application follows a *Layered Architecture*.

#### High-Level Layers

`[ Transport Layer ] -> [ Domain Layer ] -> [ Infra Layer ]`

### Layer Responsabilities

#### Transport Layer (`transport/`)

* Exposes REST endpoints

* Handles request validation

* Delegates business logic to domain layer

* Returns HTTP responses

*Annotations Used:*

* `@RestController`

* `@RequestMapping`

* `@PostMapping`

* `@GetMapping`

* `@DeleteMapping`

### Domain Layer (`domain/`)

* Contains business logic & domain objets

* Orchestrates repository calls

* Handles transactions

*Annotations Used:*

* `@Service`

* `@Transactional`

* `@EqualsAndHashCode`

### Infra Layer (`infra/`)

* Data access layer

* Interfaces extending `JpaRepository` or similar

* Query definitions

*Annotations Used:*

* `@Repository`
